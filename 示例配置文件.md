# 所有支持数据集的配置文件模板

## 分子数据集（回归任务）

### 1. ZINC数据集
```yaml
# configs/Mamba/zinc-custom-EX.yaml
out_dir: results
metric_best: mae
metric_agg: argmin
dataset:
  format: PyG-ZINC
  name: subset  # 或 full
  task: graph
  task_type: regression
  node_encoder: True
  node_encoder_name: TypeDictNode
  node_encoder_num_types: 28
  edge_encoder: True
  edge_encoder_name: TypeDictEdge
  edge_encoder_num_types: 4
prep:
  use_custom: True      # 关键：启用自定义预处理
  custom_split: False
model:
  type: GPSModel
  loss_fun: l1
gt:
  layer_type: CustomGatedGCN+Mamba_Hybrid_Degree_Noise
  layers: 4
  dim_hidden: 64
gnn:
  dim_inner: 64
optim:
  base_lr: 0.001
  max_epoch: 200
seed: 42
```

## 通用模型配置选项

### Graph-Mamba变体
```yaml
gt:
  layer_type: CustomGatedGCN+Mamba_Hybrid_Degree_Noise          # 主推模型
  # layer_type: CustomGatedGCN+Mamba_Hybrid_Degree_Noise_Bucket  # 大图版本  
  # layer_type: CustomGatedGCN+Mamba_Hybrid_Noise                # 仅排列版本
  # layer_type: CustomGatedGCN+Mamba                             # 基础版本
```

### GraphGPS基线
```yaml
gt:
  layer_type: CustomGatedGCN+Transformer  # GPS基线
```

### 纯GCN基线
```yaml
model:
  type: custom_gnn
gnn:
  layer_type: gatedgcnconv  # 或 gineconv
```
